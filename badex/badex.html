Bad Example

This Daily Jobs website is not served through HTTPS. The ensure data in transit is encyrpted, a valid TLS/SSL certificate should be used. This prevents numerous attacks such as the man-in-the-middle attack. The server header value can be changed or removed. Additionally, numerous HTTP Headers configurations are missing: Content-Security-Policy, X-Frame-Options, X-Content-Type-Options, Referrer-Policy, and Permissions-Policy.
