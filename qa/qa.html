Quality Assurance

Measures:

This site follows the team standards specified by Assignment 4 and the Final Assignment. It is version controlled in Github and deployed through Netlify. Continous integration is used and code is pushed to Github's main branch frequently. A modern build tool is used to create this site, utilizing module bundling and SASS. The site is checked through the W3C HTML and CSS Code Validator and JSHint for JavaScript files. Chrome Developer tools is used for ad hoc testing to make sure the elements and css are properly rendering on the page and also device emulation to check for responsiveness and device compatibility. Other modern dev tools used: axe DevTools, Lighthouse. Check My Links Chrome extension is used to check if all the website links are working and there are no broken links.

Additional Measures in the Future:

JavaScript Linter, Code Review, Unit Testing, End-to-End Testing
